## 🌎 Entering "The World" (Global Hub)
The newest evolution of BlackHat Pro. Access the global grid with one button.
*   **Decentralized Identity:** Create your hacker profile with just a **Username and Secret Key**. No emails, no trackers—just pure identity.
*   **Global Game Vault:** Upload and stream `.swf` files directly from the cloud. Access a community-curated library that never dies.
*   **The Comm-Link (Chat):** Engage in real-time **Global Chat** with other users or switch to **Private DMs** for 1-on-1 strategy.
*   **Hacker Network:** Send friend requests, build your contact list, and customize your identity with a **Cropped Profile Avatar**.

---

## 🕹️ Advanced Hybrid Input System
Stop fighting with basic touch controls. BlackHat Pro bridges the gap between glass and keyboard.
*   **Joystick Mode:** High-precision virtual stick for 360° movement and cursor navigation.
*   **Direct Touch:** The screen becomes a high-DPI trackpad. Tap and slide for instant, latency-free cursor response.
*   **Super Mode (Hybrid):** The ultimate setup. The **Joystick** handles Keyboard events (WASD/Arrows) while your **Touch** controls the mouse. Designed for strategy and action-RPGs.

---

## 🛠️ Custom Layout Engine 2.0
Don't settle for fixed buttons. Build your own cockpit.
*   **Visual Editor:** A drag-and-drop environment to place buttons and joysticks exactly where your thumbs need them.
*   **Precision Mapping:** Map controls to Mouse (L/M/R), any Keyboard key (A-Z, 0-9), or complex macros.
*   **Layout Sharing:** Proud of your setup? Upload your control mapping to **The World** so other players can download and use it for the same game.

---

## 🚀 Performance & Optimization
Optimized for the modern mobile web, with safety nets for older hardware.
*   **Dual-Engine Architecture:** Toggle between **Legacy (v2021)** for maximum stability and **Modern (v2026)** for the latest ActionScript 3 features.
*   **Turbo Mode:** Instantly strips heavy realism effects and post-processing to maximize FPS on resource-constrained devices.
*   **IndexedDB Binary Caching:** Global games are only downloaded once. After that, they boot instantly from your local high-speed database.
*   **⚠️ Performance Note:** Flash-to-WASM conversion is resource-heavy. While we optimize for 3GB RAM devices, high-end hardware is recommended for the best experience.

## 🛠️ Build Guide

### Prerequisites
*   [Node.js](https://nodejs.org/) & [Cordova CLI](https://cordova.apache.org/)
*   Android SDK & Build Tools

### Deployment
1. **Clone & Install:**
   ```bash
   git clone https://github.com/Anikthedev-Official/BlackHat.git
   cd BlackHat
   npm install
   ```
2.  **Initialize Android:**
    `cordova platform add android`
3.  **WASM Optimization:**
    `cordova plugin add cordova-plugin-ionic-webview`
4.  **Assemble APK:**
    `cordova build android`
